# Labels

## Priority
- P0: Critical path, blocks release
- P1: High, needed for current PI
- P2: Medium, nice-to-have this PI
- P3: Low, backlog

## Severity (bugs)
- Sev-1: Crash/data loss/corruption
- Sev-2: Broken feature/major regression
- Sev-3: Degraded UX/minor regression
- Sev-4: Cosmetic/docs only

## Work type
- feat: feature
- bug: defect
- chore: infra/maintenance
- docs: documentation
- perf: performance
- test: testing/CI
